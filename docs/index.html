Everything you need

Built for fast, clean image prep

One focused tool that handles the tedious parts of getting images ready to publish.

Per-image auto-enhance

Analyzes each photo individually and fixes its own white balance, contrast, and exposure — then sharpens. Or switch to Manual.

Exact target file size

Set a KB budget and a binary-search picks the highest quality that fits — perfect for page-speed limits.

Smart formats & transparency

Auto picks WebP and preserves PNG transparency automatically. AVIF, JPEG and PNG available when you want them.

Batch the whole folder

Drag in dozens of images or a full folder. Process in parallel with live per-image progress and error handling.

Presets or fully custom

One-click Web Image, Thumbnail, and Social presets — or go Custom and dial in resize, enhancement and export yourself.

Private & offline

Runs entirely on your machine — no cloud, no telemetry, no uploads. Your images never leave your computer.

Heads up: first launch needs one extra click

The apps aren't code-signed yet, so your operating system shows a one-time security prompt. It's expected and safe — here's how to get past it:

  • Windows: on the blue SmartScreen prompt, click More info → Run anyway.
  • macOS: right-click (or Control-click) the app → Open → Open. If it's still blocked, run xattr -dr com.apple.quarantine /Applications/ImagePrep.app.
  • Linux: make the file executable — chmod +x ImagePrep-*.AppImage — then double-click or run it.
